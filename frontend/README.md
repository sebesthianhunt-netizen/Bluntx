# BLVKDOT Frontend

React 19 + Vite + TypeScript + Tailwind + React Router + React Query + i18n.

## Prerequisites
- Node.js 18+
- npm or yarn or pnpm

## Install
```bash
cd Bluntx/frontend
npm install
```

## Run Dev
```bash
npm run dev
```

Open http://localhost:5173

## Build
```bash
npm run build
npm run preview
```
